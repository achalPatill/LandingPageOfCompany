Security Wizard

Security Wizard is an intelligent, automation-driven cPanel plugin that simplifies domain redirection management while significantly strengthening WordPress security through controlled, server-level rule enforcement.

It eliminates the need for manual configuration and reduces the risk of misconfigured redirects or insecure WordPress installations by applying best-practice rules safely and consistently.

Overview

Managing website redirections and WordPress security typically requires direct edits to the .htaccess file — a process that is both error-prone and risky for non-technical users. Even a small mistake can result in broken websites, redirect loops, or complete service downtime.

Security Wizard removes this complexity by providing a guided, structured interface that applies redirection and security rules automatically while maintaining full control over server integrity.

The tool is deeply integrated with cPanel’s UAPI system, ensuring accurate domain detection, safe execution, and compliance with hosting environment constraints.

What This Tool Allows Users To Do

Security Wizard empowers both administrators and end users to apply production-grade server rules without requiring command-line access or manual configuration.

  • Apply domain redirection rules to enforce consistent URL structures across websites
  • Enable essential WordPress security protections with a single action
  • Automatically manage and update .htaccess rules without overwriting existing configurations
  • Create automatic backups before any configuration change is applied

These capabilities ensure safer deployments, faster configuration, and reduced dependency on manual server management.

Key Features

1. Dynamic Domain Detection

Security Wizard dynamically fetches all domains associated with the cPanel account using UAPI, eliminating the risk of applying rules to incorrect paths or document roots.

  • Main domain associated with the account
  • Addon domains configured under the account
  • All active subdomains

2. Redirection Options

The tool provides standardized redirection options that cover the most common SEO and security requirements.

  • domain.com (remove www)
  • www.domain.com
  • https://domain.com
  • https://www.domain.com

Each rule is implemented using Apache mod_rewrite, enforced as a 301 permanent redirect, and automatically replaces outdated rules to prevent redirect chains and conflicts.

Automatic .htaccess Backup

To protect existing configurations, Security Wizard performs a complete validation and backup process before modifying the .htaccess file.

  • Verifies whether .htaccess exists in the document root
  • Creates a one-time backup file named .htaccess.backup-first
  • Ensures the backup file remains readable and untouched
  • Applies secure file permissions (0644)

This approach guarantees that administrators can safely roll back changes and prevents irreversible configuration loss.

WordPress Security Features

Security Wizard includes multiple WordPress hardening rules designed to block common attack vectors without affecting site functionality.

  • Block xmlrpc.php – prevents brute-force and amplification attacks targeting XML-RPC endpoints
  • Protect .htaccess / .htpasswd – blocks public access to sensitive server configuration files
  • Disable directory browsing – prevents exposure of file listings to visitors
  • Block sensitive WordPress files such as:
    • wp-config.php
    • readme.html
    • license.txt
  • Basic bot protection – blocks suspicious user agents commonly associated with automated scanning

Workflow

Security Wizard follows a structured execution flow to ensure accuracy, safety, and predictable results.

1

User selects the target domain.

2

User chooses the required redirection type.

3

User enables desired WordPress security protections.

4

User applies the configuration.

5

The tool validates paths, creates backups, removes outdated rules, applies new configurations, updates permissions, and confirms success.

Security Measures Implemented

  • Prevents duplicate rule insertion
  • Uses named comment blocks for safe rule removal
  • Ensures controlled rewrite rule placement
  • Validates document root before execution
  • Applies secure file permissions (0644)
  • Blocks unauthorized file access

Security Wizard delivers immediate operational benefits for hosting providers, administrators, and end users.

  • Eliminates manual server configuration tasks
  • Reduces configuration errors and downtime risk
  • Improves WordPress security instantly
  • Saves administrative time
  • Provides a clean, beginner-friendly interface
  • Ensures organized and maintainable rule blocks
