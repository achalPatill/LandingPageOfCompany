FAQ

What is JavaCP?

JavaCP is a professional cPanel plugin that allows hosting providers to run Apache Tomcat under individual cPanel accounts with dedicated instances per user.

How does JavaCP differ from shared Java hosting?

Unlike traditional shared Java setups, JavaCP ensures each user gets their own Tomcat instance, resulting in better performance, security, and isolation.

Do I need SSH access to manage JavaCP?

No SSH required ‚Äî full Tomcat lifecycle management (start/stop/restart) is available directly from the cPanel UI.

Can I use JavaCP with standard hosting?

Yes, JavaCP supports flexible domain mapping, allowing you to run Java applications via Tomcat and use standard cPanel hosting (HTML, JS, PHP) on the same account.
